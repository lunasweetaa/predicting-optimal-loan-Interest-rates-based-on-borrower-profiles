# predicting-optimal-loan-Interest-rates-based-on-borrower-profiles
 Proyek ini bertujuan untuk membangun model prediktif yang dapat memperkirakan tingkat bunga pinjaman yang optimal berdasarkan profil peminjam. Dengan menggunakan data peminjam yang mencakup berbagai variabel seperti skor kredit (FICO), rasio penggunaan kredit, dan jumlah cicilan bulanan, model ini akan memberikan estimasi bunga pinjaman yang sesuai dengan risiko dan kemampuan bayar peminjam.
